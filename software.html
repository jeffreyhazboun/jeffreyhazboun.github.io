Software Development

Modern astrophysics depends on large data sets,
requiring sophisticated software for analysis.
Many of my projects are software forward,
developed as tools to benefit the entire community.

hasasia

A Python package for calculting pulsar timing array sensitivity curves and sensitivity sky maps. Signal-to-noise ratios can be caluclated for stohcastic and deterministic gravitational wave sources.

  • Documentation

La Forge

A Python package for processing Bayesian analysis statistics, particulary from the PTA data processing suite, enterprise.

  • Documentation

Pulsar Data Toolbox

A Python package for building and manipulating PSRFITS data standard files.

  • Documentation

Pulsar Signal Simulator

The NANOGrav Pulsar Signal Simulator is an end-to-end simulation of pulsar signals that includes emission, propagation and instrumental affects on the radio pulses.
